#  Aura-Orb

**AI-Powered Mental Health Support Platform**

An intelligent mental health companion app leveraging Google's Gemini API to provide personalized, empathetic support. Aura-Orb combines cutting-edge AI with a user-centric design to create a safe, accessible space for mental wellness.

---

##  About Aura-Orb

Aura-Orb is a full-stack web application designed to help users navigate mental health challenges with intelligent, AI-powered conversations. The app provides:

- Intelligent Conversations: Powered by Google Gemini API for empathetic, contextual mental health support
- Privacy-First Design: Secure API integration with data privacy best practices
- User-Friendly Interface: Modern React UI for seamless interaction
- Accessible Support: 24/7 availability for mental wellness guidance

---

##  Key Features

- AI Mental Health Assistant: Intelligent chatbot providing supportive conversations and coping strategies
- Privacy-Focused Architecture: End-to-end secure communication using Google Gemini API
- Responsive Design: Works seamlessly on desktop and mobile devices
- Real-Time Conversations: Fast, interactive responses using modern web technologies

##  Quick Start

### Prerequisites
- Node.js 16+ installed
- Google Gemini API key (free tier available)

### Installation

1. Clone the repository:
git clone https://github.com/nqobile-x/Aura-Orb.git
cd Aura-Orb

2. Install dependencies:
npm install

3. Configure API Key:
- Create a `.env.local` file in the root directory
- Set your Gemini API key:
  ```
  VITE_GEMINI_API_KEY=your_gemini_api_key_here
  ```
- Get your free API key: [Google AI Studio](https://aistudio.google.com)

4. Run locally:
npm run dev
- Open `http://localhost:5173` in your browser
